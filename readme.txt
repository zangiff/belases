Лендинг средней сложности с использованием JQuery и всплывающих tooltip-ов (через bootstrap.bundle.min.js), сверстанный с применением Bootstrap 4.3.1, Gulp и препроцессора SASS. Верстка полностью адаптивна под разрешения 320px - 1920px. Ширина контейнера - 1200px, паддинги с обеих сторон - по 15px
Все сторонние библиотеки подключены с помощью CDN
